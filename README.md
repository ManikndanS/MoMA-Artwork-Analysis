# MoMA Artwork Analysis Dashboard

## Project Overview

This project analyzes the Museum of Modern Art (MoMA) artwork collection using SQL, Excel, and Power BI.

## Objectives

* Analyze artwork creation trends
* Identify the most represented artists
* Study acquisition patterns
* Explore artwork classifications and mediums
* Analyze artist nationalities and gender distribution

## Dataset Information

* Dataset: MoMA Artwork Collection
* Total Records: 157,000+
* Data Cleaning: Null values, duplicates, and invalid records handled

## Tools Used

* SQL Server
* Excel
* Power BI
* DAX

## Data Cleaning

* Removed duplicate records
* Handled null values
* Standardized column formats
* Created derived columns such as Decade

## Key Business Questions

1. How modern are the artworks at the museum?
2. Which artists are featured the most?
3. What are the most common artwork classifications?
4. What are the acquisition trends over time?
5. How diverse is the collection by nationality and gender?

## Dashboard Features

* KPI Cards
* Artist Analysis
* Classification Analysis
* Medium Analysis
* Acquisition Trend Analysis
* Nationality Analysis
* Interactive Filters 

## Key Insights

* Identified the most represented artists
* Analyzed artwork creation trends by decade
* Explored acquisition patterns across years
